# 42 ft_printf

`int ft_printf(const char *argv, ...);`

Recode printf.

- Conversions
  - c- char
  - s- string
  - p- pointer
  - d- int
  - i- int
  - u- unsigned int
  - x- hex
  - X- hex (large alpha)
  - %- percent char
- Flags
  - 0
  - \*
  - width
  - precision

## Usage example

```zsh
git clone https://github.com/Masaya-Kamei/ft_printf.git --recursive;
cd ft_printf/sample;
./run_sample.sh;
```
